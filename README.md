# Basic TodoApp REACT.JS

I Make a simple "TodoApp" with React.js.


### List of components for this project:
> - [x] LocalStorage
> - [x] DarkMode
> - [x] Add, Edit, Delete, Completed(Check)
> - [x] Responsive For Mobile
> - [-] Extention for browser
